# feat(wallet): Core Wallet MVP — FRB bridge + auth + create/import + backup

- **时间**: 2026-03-04 10:19
- **类型**: feat
- **模块**: bridge, storage, auth, wallet, backup, routing, test

## 变更意图

实现核心钱包 MVP 的完整用户流程：打开 App → 设置密码 → 创建/导入钱包 → 备份助记词 → 验证助记词 → 解锁。通过 flutter_rust_bridge v2 将 Flutter UI 层与 Rust 密码学核心连接，实现 4 层架构（domain/data/application/presentation）的 auth、wallet、backup 三个功能模块。

## 变更内容

### FRB v2 Bridge (Task 1)

- `flutter_rust_bridge.yaml` — 新增 FRB v2 配置文件，指定 Rust input/output 目录
- `rust/src/api/crypto_api.rs` — 添加 `#[flutter_rust_bridge::frb(sync)]` 宏标注
- `rust/src/api/wallet_api.rs` — 添加 FRB 宏标注
- `rust/src/api/transaction_api.rs` — 添加 FRB 宏标注
- `rust/Cargo.toml` — 启用 `ffi` feature
- `rust/src/frb_generated.rs` — FRB 自动生成的 Rust 端桥接代码
- `lib/src/rust/` — FRB 自动生成的 Dart 端桥接代码（frb_generated.dart, api/*.dart）
- `lib/main.dart` — 添加 `RustLib.init()` 初始化

### Secure Storage (Task 2)

- `lib/src/core/services/secure_storage_service.dart` — 封装 flutter_secure_storage，提供钱包凭据的加密存储/读取/删除接口

### Auth Feature (Tasks 3-5)

- `lib/src/features/auth/domain/auth_state.dart` — freezed 定义 AuthState（loading/firstLaunch/locked/unlocked）
- `lib/src/features/auth/data/auth_repository.dart` — 认证仓库，判断首次启动、密码验证（调用 Rust Argon2id）
- `lib/src/features/auth/application/auth_notifier.dart` — Riverpod 状态机，管理认证生命周期
- `lib/src/features/auth/presentation/lock_screen.dart` — 锁屏界面，密码输入 + 解锁
- `lib/src/features/auth/presentation/set_password_screen.dart` — 设置密码界面，最少 8 位 + 确认

### Wallet Feature (Tasks 6-7)

- `lib/src/features/wallet/domain/wallet_model.dart` — freezed 定义 Wallet 领域模型
- `lib/src/core/database/daos/wallets_dao.dart` — drift DAO，钱包表 CRUD
- `lib/src/features/wallet/data/wallet_repository.dart` — 钱包仓库，调用 Rust 生成助记词 + Argon2id 加密 + AES-GCM 存储
- `lib/src/features/wallet/application/wallet_notifier.dart` — Riverpod provider，钱包列表状态管理
- `lib/src/features/wallet/presentation/wallet_list_screen.dart` — 钱包列表界面
- `lib/src/features/wallet/presentation/create_wallet_screen.dart` — 创建钱包界面（名称 + 12/24 词选择）
- `lib/src/features/wallet/presentation/import_wallet_screen.dart` — 导入钱包界面（助记词输入）

### Backup Feature (Task 8)

- `lib/src/features/backup/presentation/show_mnemonic_screen.dart` — 展示助记词界面（3 列网格 + 编号）
- `lib/src/features/backup/presentation/verify_mnemonic_screen.dart` — 验证助记词界面（多选按钮 UX，随机抽 3 个位置，每个 3 选 1）

### Routing (Task 9)

- `lib/src/routing/app_router.dart` — 添加 auth guard redirect 逻辑（首次启动 → 设密码，已有钱包 → 锁屏，已解锁 → 钱包列表）

### Tests (Tasks 10-11)

- `test/unit/core/services/secure_storage_service_test.dart` — SecureStorageService 单元测试
- `test/unit/features/auth/data/auth_repository_test.dart` — AuthRepository 单元测试
- `test/unit/features/auth/application/auth_notifier_test.dart` — AuthNotifier 状态机测试（5 个用例）

- `test/widget/features/auth/lock_screen_test.dart` — LockScreen widget 测试
- `integration_test/wallet_creation_test.dart` — 端到端集成测试（FRB 桥接 + 钱包创建流程）
